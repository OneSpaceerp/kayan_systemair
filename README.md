# Kayan SystemAir

SystemAir Axial Fan Sales Module for ERPNext v16.

## Installation

```bash
bench get-app https://github.com/kayan-erp/kayan_systemair
bench --site [your-site.com] install-app kayan_systemair
bench build --app kayan_systemair
```

## Features
- SystemAir Item Creator
- Price List Data Management
- SystemAir Pricing Engine
- Enhanced SystemAir Quotation
